# Proyek Analisis Data: E-Commerce Public Dataset

Dashboard interaktif untuk menganalisis data E-commerce Brazil (Olist).

## Setup Environment - Anaconda

```
conda create --name proyek-ecommerce python=3.9
conda activate proyek-ecommerce
pip install -r requirements.txt
```

## Setup Environment - Shell/Terminal

```
mkdir proyek-ecommerce
cd proyek-ecommerce
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run Streamlit App

```
streamlit run dashboard/dashboard.py
```
